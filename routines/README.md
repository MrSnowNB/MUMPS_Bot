# Routines Directory

Place `.m` MUMPS source files here. These are **read-only inputs** to the
Guardian analysis pipeline. Never modify routine source files.

## Fetching from VistA-M

Run the fetch script to pull a routine from the WorldVistA/VistA-M repo:

```bash
bash scripts/fetch_routine.sh ORQQPL1
```

## Manual Download

If the script fails, download directly from:
https://github.com/WorldVistA/VistA-M/tree/master/Packages/Order%20Entry%20Results%20Reporting/Routines

Save to `routines/ORQQPL1.m`.
